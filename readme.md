Project Description
MDCATemy is a smart, web-based learning and testing platform designed for MDCAT aspirants in Pakistan. 
It helps students prepare efficiently through personalized quizzes, performance analytics, and targeted practice.
This project demonstrates a complete DevOps lifecycle for a containerized web application. 
It features automated CI/CD using GitHub Actions, infrastructure as code using Terraform, and cloud deployment on AWS EC2.

Technologies Used
Version Control: Git & GitHub
Containerization: Docker
CI/CD: GitHub Actions
Cloud Provider: AWS (EC2)
IaC: Terraform
